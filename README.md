# 水木plus  
  
【如何使用】
1，微信小程序搜索：水木plus  

2，或者，扫描附件二维码  
![image](https://raw.githubusercontent.com/niuap/smthplus/master/resources/smplusqrcode.jpg)


【功能特点】  
1，	无需安装，用完即走  
2，	帖子摘要，方便阅读  
3，	查看用户发帖历史，关注用户  
4，	一键分享好友，图片方便保存  
5，	综合搜索，用户id，版面，帖子一个都不少（新增）
